# Citations & References

This file tracks the references and sources compiled during the research phase for Wavelength.

## Frameworks & Web Engines
- **Phaser 4**: [GitHub Repository](https://github.com/photonstorm/phaser) - Core WebGL 2D game framework.
- **Tone.js**: [GitHub Repository](https://github.com/Tonejs/Tone.js) - Browser audio synthesis and sequencing.
- **Yuka.js**: [GitHub Repository](https://github.com/Mugen87/yuka) - JavaScript game AI framework (FSMs, steering).
- **Brain.js**: [GitHub Repository](https://github.com/BrainJS/brain.js) - Lightweight in-browser neural networks.
- **Cybercore CSS**: [GitHub Repository](https://github.com/sebyx07/cybercore-css) - Specialized styling framework for retro-futuristic styling.

## Competitor & Gameplay References
- **Zachtronics (TIS-100, Shenzhen I/O)**: [Official Website](https://www.zachtronics.com/) - Inspiration for hacking terminals and hardware skeuomorphism.
- **Oxenfree**: [Night School Studio](https://www.nightschoolstudio.com/oxenfree) - Concept reference for radio-tuning mechanics.
- **Taajuus (Hamatti/taajuus)**: [GitHub Repository](https://github.com/Hamatti/taajuus) - Multiplayer web adapter for the board game Wavelength.
